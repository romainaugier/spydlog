# spydlog

Python bindings for [spdlog](https://github.com/gabime/spdlog) using [nanobind](https://github.com/wjakob/nanobind).

## Documentation

Please refer to the official documentation of [spdlog](https://github.com/gabime/spdlog/wiki) and see [example.py](https://github.com/romainaugier/spydlog/blob/main/example.py) for examples.
